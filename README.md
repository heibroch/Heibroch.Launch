# Heibroch.Launch
A .NET shortcut bar that supports plugins


Various shortcut types...

A regular shortcut...
My Text file;"C:\MyFile.txt"

Sending an E-mail...
IT supports E-mail;"mailto:itsupport@somecompany.com"

With args...
My awesome shortcut with args; Some path [Arg=MyFirstArg] that has a property [Arg=MySecondArg]

Searching for shortcuts in other paths (e.g. shared shortcuts)...
[SearchPath];"G:\My Drive"
